# ui/

shadcn/ui primitives and shared low-level components.

## Files
- `alert-dialog.tsx` — `AlertDialog` confirmation modal with cancel/action buttons
- `auto-suggest-combobox.tsx` — `AutoSuggestCombobox` input + dropdown: type-to-filter existing values or enter new; used for equipment/muscle-group fields
- `auto-suggest-combobox.test.tsx` — tests for `AutoSuggestCombobox` (11 cases: filtering, selection, custom input, keyboard nav, accessibility)
- `badge.tsx` — `Badge` inline status/label pill
- `button.tsx` — `Button` with variant and size props
- `card.tsx` — `Card`, `CardHeader`, `CardTitle`, `CardDescription`, `CardContent`, `CardFooter`
- `checkbox.tsx` — `Checkbox` toggle input
- `command.tsx` — `Command` cmdk-based command palette primitives (`CommandDialog`, `CommandInput`, `CommandList`, `CommandGroup`, `CommandItem`)
- `dialog.tsx` — `Dialog` modal overlay
- `dropdown-menu.tsx` — `DropdownMenu` popover menu
- `input.tsx` — `Input` text field
- `label.tsx` — `Label` form label
- `popover.tsx` — `Popover` floating content panel
- `select.tsx` — `Select` dropdown with `SelectTrigger`, `SelectContent`, `SelectItem`
- `sheet.tsx` — `Sheet` slide-over panel (used by mobile nav)
- `skeleton.tsx` — `Skeleton` loading placeholder
- `tabs.tsx` — `Tabs`, `TabsList`, `TabsTrigger`, `TabsContent`
- `textarea.tsx` — `Textarea` multi-line input
